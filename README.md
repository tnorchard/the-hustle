# The Hustle

Website for **The Hustle**, a 501(c)(3) nonprofit teaching entrepreneurship and business skills to students in underserved neighborhoods.

Built with React, Vite, TypeScript, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy to Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repository.
3. Vercel auto-detects Vite — no extra config needed (`vercel.json` is included).
4. Click **Deploy**.

Or use the CLI:

```bash
npm i -g vercel
vercel
```
